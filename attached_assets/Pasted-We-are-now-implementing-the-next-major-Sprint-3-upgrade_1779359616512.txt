We are now implementing the next major Sprint-3 upgrade for the ResearchHub platform.

Current platform already includes:

* JWT authentication
* role-based collaboration
* project workspaces
* manuscripts
* datasets
* experiments
* collaboration invites
* real-time chat
* realtime manuscript editing
* citation manager
* contributor analytics
* AI research copilot

Do NOT refactor existing architecture.

This task is to implement THREE fully connected advanced systems together:

1. Peer Review Workflow System
2. Reproducibility Tracking System
3. AI Writing Assistant

IMPORTANT:
Implementation must be:

* end-to-end
* fully connected frontend ↔ backend
* stable and production-like
* resilient to API failures
* resilient to websocket disconnects
* modular
* visually impressive
* lightweight enough for Replit

Do NOT build overly complex infrastructure.
Do NOT use heavy ML training.

All APIs must:

* validate requests
* handle failures gracefully
* never silently fail
* always return proper JSON responses
