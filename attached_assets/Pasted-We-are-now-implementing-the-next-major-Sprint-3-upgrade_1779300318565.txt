We are now implementing the next major Sprint-3 upgrade for the existing ResearchHub platform.

Current system already includes:

* FastAPI backend
* React + Vite frontend
* JWT authentication
* role-based collaboration
* project workspaces
* manuscripts
* datasets
* experiments
* invites
* collaborator permissions
* real-time chat
* AI research assistant sidebar

Do NOT refactor existing architecture.

The goal is to implement THREE connected advanced features together:

1. Lightweight Real-Time Manuscript Collaboration
2. Advanced Citation Management
3. Contributor Tracking & Authorship Analytics

These features must work end-to-end and integrate naturally with the current manuscript workflow.

IMPORTANT:
Implementation should be:

* lightweight
* modular
* visually impressive
* feasible quickly
* demo-friendly

Do NOT build Google Docs-level complexity.
Do NOT use heavy CRDT infrastructure.
